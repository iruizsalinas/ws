# Contributing

Bug reports, fixes, tests, and ideas are all welcome.

## Reporting Bugs

Open an issue with what happened, what you expected, and a code example if possible. Include your Lua version and OS.

## Submitting Changes

1. Fork the repo and create a branch.
2. Run the tests before submitting:

```bash
cd tests
lua run_all.lua
```

3. Keep pull requests focused on one thing.
4. Write a short description of what you changed and why.

## Questions

If something is unclear, open an issue. Happy to help.
